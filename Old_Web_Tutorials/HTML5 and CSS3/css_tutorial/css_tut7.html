A <header> tag is important for getting ranked and designing a more descriptive web page. Here you would put something descriptive throughout all your websites, like the logo which should be the same on all the pages. It is good practice to have the <nav> inside the header.
The <nav> tag is used for adding navigation into the website. for example, for placing the following links:
  • Home
  • About
  • Contact
  • Some other links
