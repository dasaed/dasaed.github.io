Inside of the <table> tag we include a <tr> which stands for table row. Inside the <tr> tag, the first row should have <th> tags, which stand for table headers. After the <th> tag, all rows after it should have a <td> tag, which stands for table data. the <td> tags will ultimately represent the data to be filled inside the table.

Forms and form elements

Inside of the <form> tag, the following attributes must be added: <form action="" method="" >.

action is used to determine where you want to send the data to, generally a smart file, like a php file.

method is basically asking how you want to send the data. There are 2 ways to send the data: "post" and "get".
post is more secured, the user can't see it, and you can send more information through it. get actually sends the data at the end of the URL. Get is not really recommended.


There are many types of elements you can put in forms, for example:

  • text
  • password

Elementes inside the form: <input type="type of data" name="mandatory name for the element" maxlength="max amount of characters " value="default value if you want one" size="max size of the text fiel">
