variables in JavaScript must follow certain rules, such as:

  1. All variables must start with a capital, lower case or underscore only
  2. No spaces are allowed in variable names, just use an underscore
  3. Before a variable can be used, it must be first declared
  4. No JavaScript keywords are allowed to be used for variables, for example: var, function, etc.
  5. Variables are case sensitive
  6. Variable names are called identifiers

Data Types in JavaScript

There are many different types of data types:

  • Integer: var a = 21;

  • String: var a = "he said, \"something\" "
